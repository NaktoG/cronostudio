# AgentOS Enforcement Checklist

Esta checklist es obligatoria para todas las tareas.

## Checklist
1. Identificar el tipo de tarea.
2. Activar agentes pertinentes (maximo 5).
3. Auto-invocar skills aplicables por trigger.
4. Si no hay skill aplicable, explicarlo en la salida.
5. Reportar en la respuesta final:
   - Agentes activados.
   - Skills activadas.
6. Incluir resumen, archivos tocados, decisiones, riesgos y proximo paso.
