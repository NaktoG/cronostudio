# Atonix • CronoStudio — Resumen

CronoStudio es la suite de producción de Atonix para creadores de YouTube. Integra un pipeline claro (idea → guion → publicación), automatización vía n8n y analíticas diarias.

## Destacado
- Local‑first y sin dependencias externas obligatorias.
- OAuth YouTube con refresh automático.
- Dashboard de producción y disciplina semanal.
- Workflows n8n listos para sync y analytics.

## Stack
- Next.js 16 + TypeScript
- PostgreSQL 16
- n8n + Docker Compose
- Zod + JWT

## Estado actual
- Core listo: dashboard, ideas, producción, canales.
- Automatización y analytics funcionando en local.
